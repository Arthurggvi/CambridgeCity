CambridgeCity / 寒武新纪 — Windows 本地游玩包说明
================================================

【本包性质】
本压缩包为 Windows 环境下的本地游玩分发包：解压后通过自带启动脚本拉起本地 HTTP 服务，在浏览器中打开游戏。请勿直接双击 index.html 使用 file:// 打开（会导致资源加载失败）。

【如何启动】
1) 解压本包到任意目录（路径中尽量避免特殊字符）。
2) 双击根目录下的「启动寒武新纪.bat」。
3) 启动器会拉起本地同源服务并打开游戏入口页；默认使用 5511 端口；若端口被占用，启动器会报错而不会静默改端口。
4) 若以独立应用窗口打开，关闭该窗口后会回收本次本地服务；若退回普通浏览器，服务可能仍在后台，请双击「关闭寒武新纪.bat」手动回收。
5) 更细的启动与回收说明见 launcher/README_入口封装说明.md。

【包内 Node.js 运行时】
本包内的 Node.js 可执行文件（位于 launcher/runtime/node/，由发布流程在本地机器上嵌入）仅用于启动器附带的本地静态服务环境，不是游戏逻辑在 Node 里运行；游戏本体在浏览器中执行。
